# MATLAB Legacy Archive

This directory preserves the original MATLAB prototype and reference scripts used before the Python migration.

The active project is now the Python package under `src/green_vigilance/`. Normal install, test, and simulation workflows do not require MATLAB.

The original report is kept in `docs/Report_distribution_project.pdf` so project documentation lives outside the active source tree.

Some archived files may be incomplete or scaffold-like. Treat this folder as historical reference material, not as the maintained implementation path.
